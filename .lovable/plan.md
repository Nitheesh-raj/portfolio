

# Personal Portfolio Website for Nitheesh Raj M

A clean, modern, recruiter-friendly portfolio website with dark/light theme support.

## Design Direction
- Minimal, professional aesthetic with clean typography
- Dark/light mode toggle for a developer-friendly feel
- Smooth scroll navigation with a sticky header
- Fully responsive layout for mobile and desktop
- Subtle hover effects only — no heavy animations

## Sections

### 1. Navigation Bar
- Sticky top nav with links: About, Skills, Projects, Experience, Education, Contact
- Dark/light mode toggle button
- Mobile hamburger menu

### 2. Hero Section
- Large heading: **Nitheesh Raj M**
- Subtitle: **Full Stack Developer**
- Tagline: "Building responsive web applications using MERN stack and Java."
- Two CTA buttons: **View Projects** (scrolls to projects) and **Contact Me** (scrolls to contact)

### 3. About Me
- Brief paragraph about being a motivated MERN stack fresher passionate about clean code, responsive web apps, REST APIs, and continuous learning

### 4. Skills

  - **Frontend**: HTML, CSS, JavaScript, React.js, Redux, Tailwind CSS
  - **Backend**: Node.js, Express.js, MongoDB, PostgreSQL, REST APIs
  - **Programming**: Java
  - **Tools**: Git, GitHub, Postman, Firebase, Vercel, Render
- Each skill shown as a clean badge/chip

### 5. Projects

- Each card with tech tags and brief description

### 6. Internship / Experience

### 7. Education

### 8. Contact Section
- Email, phone number displayed

### 9. Footer
- Copyright line: © 2025 Nitheesh Raj M
- Social media icons (LinkedIn, GitHub, Email)

## Technical Approach
- Single-page React app with smooth scroll navigation
- Dark/light theme using `next-themes` (already installed)
- Tailwind CSS for all styling
- Component-based structure with one component per section
- Fully static — no backend needed
